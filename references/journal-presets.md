# Journal Figure Presets

These presets are practical defaults for figure preparation when official instructions are unavailable. They are not a substitute for current journal author guidelines.

## Common Defaults

| Figure type | Minimum dpi | Preferred final format |
| --- | ---: | --- |
| Photo/halftone | 300 | TIFF |
| Combination art | 600 | TIFF |
| Line art | 1200 | EPS/PDF if accepted, otherwise TIFF |

Common width classes:

| Width class | Width |
| --- | ---: |
| Small/single column | 85-90 mm |
| Medium/1.5 column | 120-140 mm |
| Large/double column | 170-180 mm |

## Publisher And Journal Starters

### Nature Portfolio

- Typical raster minimums: 300 dpi for photographs, 600 dpi for combination art, 1200 dpi for line art.
- Use RGB unless specific production instructions request otherwise.
- Common width choices: 89 mm single column, 183 mm double column.
- Prefer TIFF for raster images; preserve vector art when accepted.

### Science / AAAS

- Typical raster minimums: 300 dpi for photographs, 600 dpi for combination art, 1200 dpi for line art.
- Use high-quality TIFF, EPS, or PDF depending on figure type and submission system.
- Verify final width and maximum height against current AAAS figure-preparation instructions.

### Cell Press

- Typical raster minimums: 300 dpi for color/grayscale images, 600 dpi for combination art, 1200 dpi for line art.
- Common final widths include single-column, 1.5-column, and full-page variants; verify the exact article type.
- Use TIFF for raster images and editable/vector formats when requested by production.

### PLOS

- Typical raster minimums: 300 dpi for photographs, 600 dpi for combination art, 1200 dpi for line art.
- Prefer TIFF or EPS/PDF for high-quality figures.
- Ensure figure files are named and uploaded separately according to the target PLOS journal workflow.

### IEEE

- Common final widths: one column about 88.9 mm (3.5 in), two columns about 182.9 mm (7.16 in).
- Typical minimums: 300 dpi color/grayscale, 600 dpi combination, 1200 dpi line art.
- Use TIFF, EPS, PDF, PNG, or other formats only as allowed by the specific publication.

### Elsevier

- Typical minimums: 300 dpi halftone, 500-600 dpi combination, 1000-1200 dpi line art.
- EPS/PDF is usually preferred for vector drawings; TIFF/JPEG may be accepted for images.
- Verify whether separate source files are required at revision or production stage.

### Springer Nature

- Typical minimums: 300 dpi halftone, 600 dpi combination, 1200 dpi line art.
- Use TIFF for images and EPS/PDF for vector artwork when accepted.
- Check journal-specific maximum figure size and supplementary figure handling.

### ACS

- Typical minimums: 300 dpi color/grayscale, 600 dpi combination, 1200 dpi line art.
- Common manuscript-column widths vary by journal template; verify exact width classes.
- Check whether color is free online and whether print color has special handling.

### Wiley

- Typical minimums: 300 dpi halftone, 600 dpi combination, 1200 dpi line art.
- TIFF is common for raster; EPS/PDF is common for vector.
- Confirm journal-specific figure width, color, and graphical abstract requirements.

## Compliance Language

Use these statuses in reports:

- `pass`: source and output satisfy the selected standard.
- `pass-with-warnings`: file is usable but has caveats such as preset-based requirements, minor upscaling, or unverified color mode.
- `needs-source-improvement`: final target cannot be met without a better source file, editable vector source, higher-resolution image, or journal-specific clarification.
